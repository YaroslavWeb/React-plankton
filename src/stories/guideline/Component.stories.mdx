# How create component?

### 1. File structure
  * **index.tsx** - functional component, not exported by defualt
  * **styles.tsx**[optional] - styled components for **index**
  * **/components**[optional] - subcomponents that are closely related to **index**
    * **/components/index.tsx** - export import all your subcomponents
    * **/components/styles.tsx**[optional] - repeted styles for subcomponents
